# 太空飞船射击游戏（SDL2 + CMake + VS2022）
基于C++语言与SDL2框架开发的2D太空射击小游戏，采用CMake构建，适配Visual Studio 2022开发环境，作为C++游戏开发入门实践项目，用于提升SDL2应用开发与工程化能力。

## 项目简介
本项目为经典俯视角太空射击类游戏，核心用于实践C++游戏开发基础与SDL2框架应用，通过完整实现游戏核心逻辑，掌握游戏开发关键技术点，形成可展示的工程化项目成果。

## 核心功能
- 玩家操控：实现飞船上下左右移动控制，响应键盘输入事件

- 射击系统：完成子弹发射逻辑与矩形碰撞检测（子弹与敌机、玩家与敌机）

- 敌方逻辑：实现敌机自动生成、匀速移动及被击中销毁功能

- 工程架构：搭建清晰的模块化代码结构，使用CMake完成项目构建配置

- 基础渲染：实现游戏窗口初始化、帧渲染及帧率稳定控制

## 技术栈

- 开发语言：C++

- 图形与输入框架：SDL2（含基础渲染、事件处理功能）

- 构建工具：CMake

- 开发环境：Visual Studio 2022

- 运行平台：Windows

## 操作说明
- 方向键(WASD)：控制飞船上下左右移动

- 空格键：发射子弹

- 关闭窗口/ESC：退出游戏

## 项目亮点
1. 熟练运用SDL2框架完成窗口初始化、图形渲染、事件监听等核心操作，理解游戏渲染管线原理；

2. 掌握游戏主循环设计与碰撞检测算法，具备基础游戏逻辑开发能力；

3. 使用CMake进行项目构建与配置，熟悉工程化开发流程，提升代码可维护性。
